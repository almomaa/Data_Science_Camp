# DNA Detective Lab: The Case of the Missing Camp Trophy

This is a fictional, nonviolent classroom data science activity.
Students compare a DNA-like evidence sequence with candidate sequences using Hamming distance.

## Files

- `dna_evidence.csv`: evidence sequences
- `dna_candidates.csv`: fictional candidate sequences
- `hammingDistance.m`: MATLAB function that counts differences between two equal-length sequences
- `run_dna_detective.m`: starter MATLAB script for the activity
- `dna_answer_key.csv`: instructor answer key

## Classroom storyline

The camp trophy disappeared from the display table. A water bottle was found nearby.
The lab produced a short DNA-like sequence from the bottle. Students compare it with several fictional candidate sequences.

## Student task

1. Open `run_dna_detective.m` in MATLAB.
2. Run the script.
3. Read the ranked results.
4. Explain which candidate is closest to the evidence and why.
5. Discuss why a data match is evidence, not a complete conclusion.

## Teaching note

This activity uses simplified DNA-like strings to teach pattern comparison and Hamming distance.
Real forensic DNA analysis uses validated laboratory procedures, many genetic markers, statistics, quality checks, and expert interpretation.
